==============================
Django Apricot Systemtic Utils
==============================

This is a bunch of utilities I use across apricot systematic django
projects. It includes stuff I have written, as well as many bits and
pieces written by others but collected in to this one module.

For installation instructions, see the file "INSTALL.txt" in this
directory; for instructions on how to use this application, and on
what it provides, see the file "overview.txt" in the "docs/"
directory.
